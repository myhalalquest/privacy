Kebijakan Privasi (Privacy Policy)

Terakhir diperbarui: Juni 2026

myHalalQuest ("kami", "aplikasi") berkomitmen untuk melindungi privasi pengguna. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda.

1. Informasi yang Kami Kumpulkan

Aplikasi ini dapat mengumpulkan informasi berikut:

  • Lokasi: Digunakan untuk menemukan masjid, kuliner halal, hotel, dan destinasi terdekat. Lokasi hanya diakses saat aplikasi digunakan (foreground only).
  • Data akun (jika login): BARU Jika Anda memilih untuk masuk menggunakan Google, kami mengumpulkan nama tampilan, alamat email, dan foto profil dari akun Google Anda. Data ini disimpan di server kami (Supabase) untuk mengidentifikasi akun Anda.
  • Konten yang Anda submit: BARU Jika Anda menggunakan fitur Submit Lokasi, informasi yang Anda masukkan (nama tempat, alamat, kategori, keterangan, dll.) akan disimpan di server kami beserta ID akun Anda.
  • Data penggunaan: Informasi anonim tentang fitur yang digunakan, dikumpulkan melalui layanan analitik pihak ketiga.
  • Data iklan: ID perangkat anonim untuk menayangkan iklan yang relevan melalui Google AdMob.

2. Cara Kami Menggunakan Informasi

  • Menampilkan lokasi halal terdekat berdasarkan posisi pengguna
  • Menampilkan waktu shalat yang akurat sesuai lokasi melalui AlAdhan API
  • Menampilkan arah kiblat yang tepat
  • Mengidentifikasi akun pengguna untuk fitur yang memerlukan login, seperti Submit Lokasi
  • Memproses dan memverifikasi data lokasi yang disubmit oleh pengguna sebelum ditampilkan di aplikasi
  • Meningkatkan kualitas dan performa aplikasi
  • Menayangkan iklan yang relevan

3. Penyedia Layanan Login BARU

Aplikasi ini menggunakan Google Sign-In sebagai metode login. Saat Anda masuk dengan Google, kami menerima informasi dasar profil Anda dari Google sesuai izin yang Anda berikan. Kami tidak menyimpan kata sandi Google Anda. Lihat Kebijakan Privasi Google untuk detail lebih lanjut.

Data akun disimpan dan dikelola menggunakan layanan Supabase (Kebijakan Privasi Supabase), yang berfungsi sebagai backend database aplikasi kami. Server Supabase yang kami gunakan berlokasi di wilayah Asia Tenggara.

4. Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan pihak ketiga berikut yang memiliki kebijakan privasi masing-masing:

  • Google Sign-In — layanan autentikasi login
  • Google AdMob — layanan iklan
  • Supabase — penyimpanan data akun dan lokasi
  • OpenStreetMap / Overpass API — data peta dan lokasi
  • AlAdhan API — data waktu shalat berdasarkan lokasi

5. Izin Aplikasi

  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION: Untuk fitur pencarian lokasi terdekat, kiblat, dan waktu shalat
  • INTERNET: Untuk mengambil data lokasi, login, dan menampilkan iklan
  • VIBRATE: Untuk fitur tasbih digital (haptic feedback)

6. Penyimpanan dan Keamanan Data

Untuk pengguna yang tidak login, kami tidak menyimpan data pribadi di server kami. Data lokasi hanya diproses secara lokal di perangkat.

Untuk pengguna yang login, data akun (nama, email, foto profil) dan konten submission disimpan di server Supabase kami. Kami menggunakan enkripsi standar industri untuk melindungi data Anda. Anda dapat menghapus akun Anda dengan menghubungi kami melalui email.

7. Privasi Anak

Aplikasi ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak.

8. Hak Pengguna BARU

Sebagai pengguna yang memiliki akun, Anda berhak untuk:

  • Mengakses data pribadi yang kami simpan tentang Anda
  • Meminta penghapusan akun dan data pribadi Anda
  • Menggunakan sebagian besar fitur aplikasi tanpa harus login

Untuk mengajukan permintaan terkait data Anda, silakan hubungi kami di myhalalquest@gmail.com.

9. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini sewaktu-waktu. Perubahan akan diberitahukan melalui pembaruan aplikasi. Penggunaan aplikasi secara berkelanjutan setelah perubahan berarti Anda menyetujui kebijakan yang diperbarui.
